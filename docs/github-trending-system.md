# GitHub Trending 深度分析系统

## 系统概述

本系统用于深度分析 GitHub Trending 排行榜，产出两类高质量文章：
1. **榜单综述文章** - 每日/每周/每月趋势分析
2. **单个仓库深度文章** - 深入分析热门仓库

## 核心价值

- **精准流量** - GitHub Trending 是开发者关注的热点
- **技术深度** - 深入分析技术亮点和趋势
- **实用价值** - 提供使用建议和最佳实践
- **SEO 优势** - 长尾关键词覆盖

## 系统架构

### 1. 数据采集层
```
scripts/fetch-trending.ts → 抓取 GitHub Trending 数据
scripts/data/trending-{date}.json → 保存原始数据
scripts/data/analysis-{date}.json → 保存分析结果
```

### 2. 数据分析层
- 语言分布分析
- 主题分类（AI/Agent、开发工具、安全工具、学习资源）
- 爆点识别（增长最快的仓库）
- 趋势预测

### 3. 内容生成层
#### 3.1 榜单综述文章
- **标题**: GitHub Trending 每日/每周/每月综述
- **内容结构**:
  1. 榜单概览（总仓库数、总增长 stars）
  2. 热门仓库 Top 5（亮点分析）
  3. 趋势洞察（语言分布、主题分类）
  4. 爆点解读（为什么火）
  5. 值得关注的仓库推荐
  6. 总结与展望

#### 3.2 单个仓库深度文章
- **标题**: 深度解析：{仓库名} - {一句话亮点}
- **内容结构**:
  1. 仓库概览（简介、增长数据）
  2. 核心亮点（解决什么问题）
  3. 技术架构（关键组件）
  4. 使用指南（快速开始）
  5. 最佳实践（使用建议）
  6. 同类对比（与竞品比较）
  7. 未来展望（发展趋势）
  8. 总结

### 4. 发布层
- 添加到 `ai-tools-data.ts`
- 构建网站
- 部署到 Cloudflare Pages
